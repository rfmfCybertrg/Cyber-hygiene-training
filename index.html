Cyber Hygiene Training

Password Security

Use strong, unique passwords for each account. Avoid using easily guessable information like birthdays or common words.

Software Updates

Regularly update your software to protect against vulnerabilities. Enable automatic updates whenever possible.

Phishing Awareness

Be cautious of unsolicited emails or messages asking for personal information. Verify the sender's identity before clicking on links or downloading attachments.

Safe Internet Practices

Use secure connections (HTTPS) and avoid using public Wi-Fi for sensitive transactions. Be mindful of the websites you visit and the information you share online.

Mobile Device Security

Keep your mobile devices secure by using PINs, passwords, or biometric authentication. Install apps only from trusted sources and keep your device's software up to date.
